Instructions

1) Prepare your XML

The input for Narthex is XML which is first packed in a GZIP or ZIP archive. The uploaded archive can contain either a single XML file or a collection of files.

2) Login and Upload

Before uploading your XML you must register/login to Narthex. Once logged in, all you have to do is drag the archive and drop it on the upload page. The analysis will begin immediately.

3) Explore

When the initial analysis is complete, you will be able to explore the tree that represents the XML structure. The complete analysis continues in the background, and should be complete within a few minutes. All values are sorted, collated and counted, and histograms are made available.

4) Review and Access

When the second phase of analysis is complete, the resulting tree and terminology lists can be explored in detail, and they are then also available for other software to access via the REST API.
