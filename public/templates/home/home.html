Narthex

1) Prepare your XML

The input for Narthex is XML (either as text or GZipped text). The first step for you is to dump your data to XML and decide if you want to put it in a zip package.

2) Login and Upload

Before uploading your XML you must register/login to Narthex. Once logged in, all you have to do is drag and drop the xml source package on to the dashboard file-drop area. The analysis will begin immediately.

3) Explore

When the initial analysis is complete, you will be able to explore the tree that represents the XML structure. The complete analysis continues in the background, and should be complete within a few minutes. All values are sorted, collated and counted, and histograms are made available.

4) Review and Access

When the second phase of analysis is complete, the resulting tree and terminology lists can be explored in detail, and they are then also available for other software to access via the REST API. The API link contains a key which gives permission.

5) Think about the Potential

Once you have some experience with this program, think about what could be done with it in the future, and let us know about it! (narthex@delving.eu) It is an open source project so we can use the ideas and funding to take the project further. More graphical renderings will be possible using the D3 Library, and be sure to press the "Enrich" button to see the enrichment potentials.
